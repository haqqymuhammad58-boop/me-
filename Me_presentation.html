// 00 — VISI PROYEK

Apa itu Me?

Terinspirasi dari JARVIS milik Tony Stark — Me adalah personal AI assistant yang dibangun secara lokal, privat, dan modular. Dimulai dari fondasi sederhana, setiap versi menambahkan lapisan kecerdasan baru hingga menjadi agen otonom yang benar-benar "mengenal" pemiliknya.

🔒

100% Lokal & Privat

Semua proses berjalan di komputer sendiri. Tidak ada data yang dikirim ke cloud. Privasi adalah prioritas utama.

🧩

Modular & Skalabel

Setiap versi menambahkan modul baru tanpa merusak yang lama. Mulai dari v1 yang sederhana hingga v5 yang otonom.

🗣️

Voice-First Interface

Interaksi utama melalui suara — seperti berbicara dengan asisten nyata. Tidak perlu mengetik perintah.

🧠

AI yang Benar-benar Mengenal Anda

Membangun profil pengguna, mengingat percakapan, memahami rutinitas — dan akhirnya bertindak secara proaktif.

🚀

Menuju Agen Otonom

Goal akhir: sistem yang dapat merencanakan, mendelegasikan, dan menyelesaikan tugas kompleks secara mandiri.

⚙️

Open Source Stack

Dibangun dengan Whisper, Ollama, Llama 3, Piper TTS, dan ekosistem Python yang sepenuhnya gratis dan terbuka.

// 02 — EVOLUSI SISTEM

Roadmap Versi V2 → V5

Setiap versi membangun di atas fondasi sebelumnya — menambahkan lapisan kecerdasan, konektivitas, memori, dan otonomi baru.

V2.0

Me Cerdas — Context & Skills

Memori percakapan, pencarian web, modul skill modular, wake word detection, dan logging percakapan. Sistem mulai "mengingat" apa yang terjadi sebelumnya.

+ Memori percakapan + Web search (DuckDuckGo) + Skill modules + Wake word detection + SQLite logging
V3.0

Me Terhubung — System Integration

Kontrol OS penuh, browser automation, file manager, GUI dashboard, notifikasi push, dan integrasi API eksternal. Me kini bisa mengoperasikan komputer secara langsung.

+ Kontrol OS (pyautogui) + Browser automation + File manager + GUI dashboard + Notifikasi push + API eksternal
V4.0

Me Mengenal Anda — Memory & Vision

Long-term memory dengan Vector DB, profil pengguna personal, kemampuan melihat layar/kamera, membaca dokumen, dan respons proaktif. Me mulai benar-benar "mengenal" Anda.

+ Vector DB (ChromaDB) + User profile & routine + Vision (kamera/screenshot) + Baca PDF & dokumen + Proactive suggestions
V5.0

Me Otonom — Full AI Agent

Puncak evolusi: self-planning dengan ReAct pattern, sistem multi-agent, self-monitoring, tool use otomatis, mode server API, dan feedback loop pembelajaran mandiri. Me yang sesungguhnya.

+ Task planner (ReAct) + Multi-agent system + Self-monitoring + Tool use otomatis + API / server mode + Self-improvement loop
// 03 — ROADMAP DEVELOPMENT

Jalur Pengembangan

Dari perintah suara sederhana menuju agen AI yang benar-benar otonom — setiap milestone membangun kapabilitas di atas yang sebelumnya.

V1.0 — Fondasi (MVP)
Voice input → AI processing → voice/action output. Sistem paling dasar yang sudah bisa berbicara dan mengeksekusi perintah sederhana di komputer.
Whisper STT Ollama + Llama 3 Piper TTS Python
V2.0 — Cerdas
Tambah memori percakapan, wake word "Hey Alpha", web search, dan modul skill yang bisa diperluas. Sistem mulai "mengingat" konteks.
Context Manager DuckDuckGo API SQLite Skill Modules
V3.0 — Terhubung
Kendali OS penuh, browser automation, file management, notifikasi, dan integrasi ke layanan eksternal (Spotify, Gmail, cuaca). GUI dashboard pertama.
pyautogui Playwright Telegram Bot PyQt GUI
V4.0 — Mengenal Anda
Long-term memory dengan vector database, profil pengguna personal, kemampuan melihat layar & kamera, membaca dokumen, dan respons proaktif tanpa diminta.
ChromaDB OpenCV LlamaIndex User Profiling
V5.0 — Otonom (Goal)
Orchestrator agent dengan ReAct planning, multi-agent delegation, self-monitoring, tool use mandiri, FastAPI server mode, dan self-improvement feedback loop.
ReAct Pattern Multi-Agent FastAPI Self-Learning
